# CLAUDE.md — Enterprise AI Ledger (Meridian prototype)

A publishable, interactive prototype of the **Enterprise AI Ledger** — a Hepburn
Advisory product concept that makes AI cost-to-outcome control tangible for a
CFO/CIO, populated with a fictional financial-services company (**Meridian
Financial Group** — illustrative, not real).

- **Owner:** Simon Bowker (sibowker@gmail.com)
- **Repo:** github.com/sibowker-oss/enterprise-ai-ledger
- **Stack:** Next.js static export → GitHub Pages (sibowker-oss.github.io/enterprise-ai-ledger)
- **Not** The AI Ledger (apac-ai-intel) and **not** the Arda marketing site — a separate product.

## Publishing gate

Nothing reaches a public URL without a staging link and Simon's explicit
in-session approval. This repo's `main` builds to GitHub Pages — treat any push
to `main` as a publish and get approval first.
